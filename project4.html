"Go Run" Game Design

"Go Run" Game Design, using Blender and Unity.

Project Overview

Go Run is a captivating endless runner game where players control a character that must navigate through a maze, avoiding traps and obstacles to reach the next level. This project was developed to demonstrate the integration of various 3D animation techniques, including modeling, texturing, rigging, and animation.

Key Features

  • Dynamic Maze Generation:
    The game features dynamically generated mazes that ensure a unique experience every time the game is played.
  • Trap Mechanisms:
    Moving traps add an extra layer of challenge, requiring precise timing and strategy to avoid them.
  • Simple, Intuitive Controls:
    The game's control scheme is easy to learn but difficult to master, making it accessible to all players while providing a deep challenge.
  • Level Progression:
    As players reach the green goal zone, they progress to new, more challenging levels with different maze configurations and traps.

Process and Development

The development process followed these steps:

  • Concept and Storyboarding: The initial concept was designed around a simple, yet engaging, endless runner with a focus on maze navigation.
  • 3D Modeling: The game was modeled in Unity, starting with basic shapes like cubes for the character and walls.
  • Texturing, Rigging, and Skinning:Textures were applied to all game objects to enhance visual appeal. Rigging was implemented for the character to allow for smooth movement and animations.
  • Animation and Lighting: The character's movements were animated, and the game's environment was lit to create an immersive experience. Camera angles were carefully selected to provide the best view of the action.
  • Final Output: The game was rendered with two levels, each providing a unique challenge. The final product was tested and refined based on feedback.
